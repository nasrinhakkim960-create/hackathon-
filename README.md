# hackathon-
Remote health monitoring system

This project is a simple IoT-based Remote Health monitoring system designed for ruralband remote areas.It simulates a health device that measures patients vitals such as temperature and heart rate and sends to a doctor dashboard in real-time.

How it works:

1.Simulated device generates random health data

2.Data is send to a backend server using HTTP request

3.The backend processes the data and checks for alerts.

4.The dashboard displays real-time data and warnings and even changes color to show the danger.
